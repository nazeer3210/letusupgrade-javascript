There are 14 console functions and each of them have a particular use.

  • The console.assert() method writes a message to the console, but only if an expression evaluates to false.
    Syntax console.assert(expression,message);
  • The console.clear() method clears the console and can also be used to write a message.
    Syntax console.clear()
  • Writes to the console the number of times that particular console.count() is called.
    Syntax console.count(label)
  • The console.error() method writes an error message to the console.
    Syntax console.error(message)
  • The console.group() method indicates the start of a message group.
    Syntax console.group(label)
  • The console.groupCollapsed() method indicates the start of a collapsed message group.
    Syntax is console.gorupCollapsed();
  • The console.groupEnd() method indicates the end of a message group.
    Syntax is console.End()
  • he console.info() method writes a message to the console.
    Syntax is console.info(message)
  • The console.log() method writes a message to the console. This can also write objects and arrays to the console
    Syntax is console.log(message)
  • The console.table() method writes a table in the console view.
    Syntax is console.table(tableData,tableColumns);
  • The console.time() method starts a timer in the console view.
    Syntax is console.time(label)
  • The console.timeEnd() method ends a timer and writes the result in console view.
    Syntax is console.timeEnd()
  • The console.trace() method displays a trace that show how the code ended up at a certain point.
    Syntax is console.trace(label)
  • The console.warn() method writes a warning to the console.
    Syntax is console.warn(message)
